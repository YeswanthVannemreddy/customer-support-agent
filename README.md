\# Customer Support Agent (AI Capstone Project)



This project is a simple rule-based \*\*Customer Support Agent\*\* built using Python.  

It can:

\- Detect intent (refund, billing, cancellation, etc.)

\- Classify urgency (high, medium, low)

\- Generate automated responses

\- Store conversation memory



\## How to Run

```bash

python main.py
